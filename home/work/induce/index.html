Motion graphic video designed to showcase a prototype app UI for lucid dreaming. This app helps to make the process of lucid dreaming easy for the user, through consistent journaling, reality checks, and MILD. As the user spends more time using the app, the UI adapts to reflect their progress along their lucid dreaming journal and allows them to set more and more challenging goals.
